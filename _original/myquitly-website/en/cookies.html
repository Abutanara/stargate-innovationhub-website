2. What are Cookies?

A cookie is a simple small file that is sent along with the pages of this website and stored by your browser on the hard drive of your computer (or other device). The information stored in it can be sent back to our servers or those of a relevant third party on a later visit.

3. What are Scripts?

A script is a fragment of program code that ensures that our website functions correctly and interactively. This code is executed on our server or your device.

4. What is a Web Beacon?

A web beacon (or pixel tag) is a small, invisible fragment of text or image on a website that is used to monitor traffic on a website. To enable this, various data about you is stored using web beacons.
